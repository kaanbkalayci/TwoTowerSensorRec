# Reports

Generated figures, tables, and written summaries belong here.

## Layout

- `figures/`: exported plots from notebooks.
- `tables/`: metrics tables or compact CSV summaries.

Suggested outputs:

- Data-alignment and gap plots from notebook 1.
- Common-objective comparison tables from notebook 2.
- Expanded-vs-RSSI-only ablation tables from notebook 3.
- Final selected-model metrics and deployment notes.

The contents of `figures/` and `tables/` are ignored by Git by default, except
for `.gitkeep`.
