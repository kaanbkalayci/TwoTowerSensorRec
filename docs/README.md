# Docs

Use this directory for methodology notes, feature dictionaries, experiment
decisions, and interpretation writeups that should survive beyond notebook
outputs.

Good candidates:

- Utility-function definitions and rationale.
- Feature dictionary for baseline and expanded feature sets.
- Notes on train/validation/test split design.
- Decisions about primary metrics and deployment constraints.
- Short summaries of important experiment results.

Large generated plots and tables should go in `reports/`, not here.
